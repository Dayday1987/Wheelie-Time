# Wheelie-Time
Physics based motorcycle wheelie game

1. touch throttle to start countdown.
2. engage throttle to pop wheelie.
3. use the clutch and throttle to balance gravitykeep and torque and wheelie going.
4. applying too much throttle results in over rotating and wrecking.
5. applying too little throttle results in front tire coming back down.
6. game ends when user wrecks or front tire makes contact with the road.
